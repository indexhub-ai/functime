Installation

functime is published as Python package in PyPI called functime. To install the latest functime release, run the following command: 

pip install functime
Then login with the command functime login, which triggers an interactive login experience through your browser.

Usage Limits

functime's free tier comes with 2GiB of data uploads, 10 deployed forecasters, and up to 1 million predictions. Need an upgrade? Book a 15 minute discovery call on Calendly.
