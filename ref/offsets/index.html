offsets

freq_to_sp(freq, include_dec=False)

Return seasonal periods given offset alias.

Parameters:

Name Type Description Default
freq str

Offset alias supported by Polars.

The offset is dictated by the following string language:

  • 1ns (1 nanosecond)
  • 1us (1 microsecond)
  • 1ms (1 millisecond)
  • 1s (1 second)
  • 1m (1 minute)
  • 1h (1 hour)
  • 1d (1 day)
  • 1w (1 week)
  • 1mo (1 calendar month)
  • 1q (1 calendar quarter)
  • 1y (1 calendar year)
  • 1i (1 index count)
 required
include_dec bool

If True, return floating point seasonal periods. Otherwise, all seasonal periods are rounded down to the nearest integer.

 False

Returns:

Name Type Description
sp list of int, list of float
